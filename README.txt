
        “隐身草”是个将系统隐藏设置功能提供给用户使用的一个小工具。
          由于系统隐藏的设置功能众多，所以“隐身草”只筛选了一部分用户在其它操作系统平台中经常使用的功能，如显示隐藏文件等等。
          本软件安装之后不会产生任何的无用的文件甚至是app运行功能，待你安装成功之后将在在你的“系统偏好设置”中多出一个“隐身草”的选项，进入此项之后将可以进行一些特殊的系统设置。

版本1.1更新内容
1.添加了安全睡眠的模式；
2.添加了对登录背景的修改；

注：OS X提供了一种“安全睡眠”功能，将睡眠和休眠功能结合起来。当进入安全睡眠时，OS X首先把工作状态存储到硬盘，然后将计算机转入睡眠状态。如果睡眠中电源没有中断，则恢复时就不需要读取硬盘。如果安全睡眠的过程中电源中断了，下一次开机则会从硬盘恢复工作状态。
